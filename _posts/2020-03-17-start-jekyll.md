Useful Command lines and Steps to start the Jekyll site.

- Install DEPENDENCIES

```
bundle install
```

- Test a Site Locally

```
bundle exec jekyll serve
```

- Build a Site

```
bundle exec jekyll build
```

- Creata a new post

```
bundle exec jekyll post "My New Post"
```
